# Skating Analyzer v1.0.0

## Highlights

- Figure skating training video upload and async analysis workflow
- MediaPipe pose estimation and biomechanics metrics
- AI-generated training report pipeline
- Child mode and parent mode UX
- Skill tree, archive, plan, and progress views
- Docker all-in-one deployment

## Included in this release

- Public repository cleanup for open-source sharing
- `.env` / runtime data / exported image archives kept out of Git
- Updated README and added Chinese documentation
- Exported all-in-one image available locally as:
  - `skating-analyzer-allinone-latest.tar`

## Notes

- Fill in your own keys via `.env`
- Runtime data is stored under `./data`
- The repository does not include real credentials or user video assets
