# Skating Analyzer v1.0.0

## EN

### Highlights

- End-to-end figure skating training video analysis workflow
- Motion frame extraction, pose estimation, and biomechanics metrics
- AI-generated structured reports for training review
- Child mode and parent mode product experience
- Skill tree, archive, progress, and training plan modules
- Docker all-in-one deployment for local delivery

### Packaging

- Latest all-in-one image can be exported locally as:
  - `skating-analyzer-allinone-latest.tar`

### Notes

- Configure your own keys via `.env`
- Runtime data is stored under `./data`
- This public repository does not include real credentials, databases, or user video assets
