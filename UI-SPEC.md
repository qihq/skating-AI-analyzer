## 2. 设计系统

### 2.1 双主题系统

应用支持两套 UI 主题，通过 `data-ui-theme` 属性切换：

| 主题 | 值 | 描述 |
|------|----|------|
| 经典模式 | `classic`（默认） | 浅色背景，白色卡片，蓝色主色调 |
| 冰场暗玻璃 | `ice-glass` | 深色沉浸式界面，玻璃拟态，青色主色调 |

**切换机制：**
- `AppearanceContext` 管理主题状态
- localStorage key: `icebuddy.ui-theme`
- `<html>` 和 `.app-shell` 元素上设置 `data-ui-theme` 属性
- 所有组件通过 `[data-ui-theme="ice-glass"]` CSS 选择器应用深色覆写

### 2.3 Ice-Glass 主题覆写

深色主题下，以下 CSS 变量被覆写：

```css
:root[data-ui-theme="ice-glass"] {
  --bg-primary: #07131d;
  --parent-primary: #f8fafc;
  --parent-accent: #67e8f9;      /* 青色替代蓝色 */
  --parent-bg: #07131d;
  --parent-surface: rgba(15, 23, 42, 0.58);
  --parent-border: rgba(255, 255, 255, 0.14);
  --color-primary: #67e8f9;
}
```

**深色主题视觉特征：**
- 背景：深蓝黑 #07131d + 径向渐变光晕（青色、蓝绿色）
- 表面：半透明深色 + `backdrop-filter: blur(18-22px)`
- 卡片：`linear-gradient(135deg, rgba(15,23,42,0.68), rgba(15,23,42,0.4))`
- 边框：`rgba(255,255,255,0.13-0.14)`
- 文字：白/浅灰 #f8fafc, #dbeafe, #cbd5e1
- 强调色：青色 #67e8f9 替代经典模式的蓝色
- 内阴影：`inset 0 1px 0 rgba(255,255,255,0.09-0.16)`
- 模糊光晕：`::before` 和 `::after` 伪元素固定定位的大尺寸模糊圆形

### 4.4 PIN 验证系统

- PIN 长度：4-6 位数字，可配置
- 输入组件：`PinInput` — 独立数字格子，错误时 shake 动画
- 锁定机制：连续 3 次错误锁定 30 秒
- 使用场景：进入家长模式、删除分析记录、重新分析

## 6. 通用组件

### 6.1 ReportCard — 报告区块卡片

```tsx
<ReportCard title="总体评价" eyebrow="Summary" className="可选额外类">
  {children}
</ReportCard>
```

渲染为 `.app-card` 容器，顶部有蓝色小标签 (eyebrow) + 标题 (h2)。

### 6.2 ForceScoreRing — 分数圆环

SVG 圆环进度条，100x100 viewBox，半径 42，stroke-width 8。
- 颜色根据分数自动切换（绿/琥珀/红）
- 中心显示数字分数 + "分"
- 支持自定义尺寸 class

### 6.3 ZodiacAvatar — 生肖头像

支持三种类型：
- `zodiac_rat`: SVG 鼠头像（坦坦）
- `zodiac_tiger`: SVG 虎头像（昭昭）
- `emoji`: 圆形 emoji 容器

尺寸: sm(32px), md(48px), lg(96px), xl(120px)

可选 `animate` 属性触发 `avatar-bounce` 动画。

### 6.4 XpProgressBar — 经验值进度条

渐变进度条（violet → pink），带 shimmer 光泽动画。
- 容器: `h-3 rounded-full bg-gray-100`
- 进度: `bg-gradient-to-r from-violet-400 to-pink-400`
- 光泽: `animate-shimmer` 白色半透明斜条

### 6.5 SkillNodeCard — 技能节点卡片

根据状态显示不同颜色方案的卡片，包含：
- 状态圆点（左上角）
- Emoji 图标
- 技能名称
- 状态文字
- 进度条（attempting 状态）
- 分数显示
- 操作按钮

可选 `highlightTone="focus"` 添加金色高亮边框。

### 6.6 PinInput — PIN 输入框

独立数字格子输入，每个格子是一个单独的输入框。
- 错误时 `shake` 动画
- 支持自动聚焦

### 6.7 UiThemeToggle — 主题切换

两种变体：
- `compact`: 小按钮，显示"新 UI"/"经典 UI"
- `hero`: 大卡片样式，带描述文字和 toggle 开关

### 6.8 UnlockCelebration — 解锁庆祝

技能点亮时的全屏庆祝动画：
- 8 个方向的星形粒子爆炸 (`star-burst` 动画)
- 中心标签弹出 (`unlock-pop` 动画)

## 8. 动画与过渡

### 8.1 关键帧动画

| 名称 | 效果 | 用途 |
|------|------|------|
| `shimmer` | translateX(-100%) → translateX(400%) + skewX(-12deg) | 加载光泽、进度条光泽 |
| `float` | translateY(0) → translateY(-8px) → 0 | 浮动效果 |
| `avatar-bounce` | translateY(0) → translateY(-4px) → 0 | 头像弹跳 |
| `unlock-pop` | scale(0) rotate(-10deg) → scale(1.2) rotate(5deg) → scale(1) | 解锁弹出 |
| `shake` | translateX(-4px, 4px, -3px, 3px, 0) | 错误抖动 |
| `star-burst` | translate(var(--tx), var(--ty)) scale(0→0.5), opacity(1→0) | 星形粒子 |

### 8.2 Star-burst 方向变量

```css
.star-1 { --tx: 60px; --ty: -60px; }
.star-2 { --tx: -60px; --ty: -60px; }
.star-3 { --tx: 80px; --ty: 0px; }
.star-4 { --tx: -80px; --ty: 0px; }
.star-5 { --tx: 60px; --ty: 60px; }
.star-6 { --tx: -60px; --ty: 60px; }
.star-7 { --tx: 0px; --ty: -80px; }
.star-8 { --tx: 0px; --ty: 80px; }
```

### 8.3 无障碍

```css
@media (prefers-reduced-motion: reduce) {
  .animate-shimmer, .animate-float, .animate-avatar-bounce,
  .animate-shake, .unlock-pop, .star-particle {
    animation: none;
  }
}
```

## 9. 设计模式总结

### 9.1 视觉语言

- **毛玻璃拟态 (Glassmorphism)**: 半透明卡片 + backdrop-blur + 微妙边框
- **大圆角**: 卡片 24-32px, 按钮/标签 999px (全圆角), 输入框 20px
- **渐变背景**: body 上有微妙的径向渐变，卡片悬停有渐变变化
- **柔和阴影**: 大范围低透明度阴影 (`0 18px 50px rgba(15,23,42,0.08)`)

### 9.2 交互设计

- **最小触摸目标**: 44px (成人), 56px (儿童)
- **状态反馈**: 颜色编码徽章 (绿=成功, 琥珀=警告, 红=错误, 蓝=信息)
- **渐进式披露**: 儿童模式隐藏复杂度，家长模式展示完整细节
- **粘性定位**: 桌面端侧边栏和摘要面板使用 sticky
- **内容可见性**: 长列表使用 `content-visibility: auto` 优化性能

### 9.3 布局模式

- **移动优先**: 默认移动端布局，通过断点渐进增强
- **桌面端侧边栏**: `web` 断点 (1280px) 触发 240px 固定左侧边栏
- **两栏布局**: 多数页面在桌面端使用 `grid-cols-[比例_比例]` 两栏
- **卡片堆叠**: 页面内容由多个 `.app-card` 垂直堆叠，间距 `space-y-6`

### 9.4 组件复用模式

- 所有卡片统一使用 `.app-card` 基础类
- 所有按钮统一使用 `.app-pill` 或 `rounded-full bg-blue-500` 模式
- 所有表单统一使用 `.app-input/.app-select/.app-textarea`
- 所有状态指示统一使用 `rounded-full px-3 py-1 text-sm` 徽章模式
